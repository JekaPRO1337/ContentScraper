Complete Guide

Welcome to the official documentation for Content Scraper Premium. This tool is designed for professional content management and automated cloning between Telegram channels.

1. Quick Start

System Requirements

  • Windows 10/11 or Linux Server (Ubuntu 20.04+).
  • Python version 3.10 or higher.
  • Stable internet connection.

Installation

1. Download the content-scraper-premium.zip archive.

2. Extract it into a separate folder.

3. **Open Terminal**: To open a terminal in the folder, **right-click** on an empty space and select Open in Terminal (or Shift + Right-click -> Open PowerShell window here).

4. Run install.bat — this will automatically create a virtual environment and install all necessary libraries.

2. Credential Configuration

The script requires your personal Telegram API keys. This is safe — they are stored only on your machine.

Obtaining API_ID and API_HASH

  • Go to my.telegram.org.
  • Log in and create a new application (any name).
  • Copy App api_id and App api_hash.
  • Open the .env file and paste them into the corresponding fields.
  • Important: Quotes are optional in the .env file.

Creating a Bot

  • Message @BotFather on Telegram.
  • Create a new bot and get the API Token.
  • Open the .env file and paste the token into the BOT_TOKEN field.
  • In the ADMIN_ID field, enter your personal ID (you can find it through @userinfobot).

3. Using the Admin Panel

After starting the script (python main.py), message your bot /start. This will open an interactive menu.

Adding Channel Pairs

Click "➕ Add Pair". The bot will ask for the Donor ID and Target ID.

  • Donor: The source channel. You must be subscribed to it with your main account.
  • Target: The destination channel. The bot must be assigned as an Administrator in it.
Important: If you are using channel IDs (starting with -100...), make sure the bot "knows" these channels. Usually, adding the bot as an admin to the target is enough.

4. Advanced Features

Text Removal and Replacement (Regex)

You can set rules to replace links or words. In the admin panel, select a pair and click "⚙️ Text Settings".

Example: To remove all mentions of a competitor, add a rule: regex:.*competitor.* -> replace with [Your Brand].

Resolving PEER_ID_INVALID

Sometimes the bot cannot see the target channel by its ID. Our script implements a "learning" system:

  1. The script uses your User account to find the channel.
  2. It then passes the data to the Bot via username (e.g., @mychannel).
  3. The bot successfully caches the channel, and the error disappears forever.

5. Channel ID Sniffer (Find IDs)

It can often be difficult to find the ID of a private channel or a channel you just subscribed to. For this, we created the Sniffer Tool.

How to Start

  1. Run python main.py.
  2. In the menu that appears, select option 2 (Channel ID Sniffer).
  3. The script will start listening to all incoming messages on your account.

Viewing Results

As soon as messages appear in the desired channels, the sniffer will capture their IDs. You can see the results in real-time in the console or open the file:

sniffer_log.html

This is a beautiful HTML report where you can simply click on an ID to copy it to your clipboard for further addition to the scraper.

7. Account Switch and Session Reset

If you need to change the Telegram account linked to the script or completely reset the database, follow these instructions.

How to Reset Session

1. Completely turn off the script (close the terminal window).

2. Find files with the .session extension in the project's root folder.

3. Delete these files. The next time you start the script, it will ask for the phone number again.

How to Clear Database

The database.db file stores all your channel pairs, statistics, and button settings. If you want to start from scratch, simply delete this file.

Attention: Never delete or modify files with the .py extension unless you are a developer. This will break the script.

8. FAQ

Do I need a VPN?

If Telegram is not directly blocked in your region, no. If you run the bot from a restricted region without a proxy, there may be delays.

Video Note Support (Round Videos)

The Premium version includes automatic conversion of videos into the Video Note format. The bot will automatically crop the video to the required aspect ratio and send it as a round video message.

  • Pro License: Required for the conversion feature to work.
  • Automatic Processing: If you have a valid license, the bot detects and handles video notes automatically.

How many channels can I add?

There are no limits in the Premium version, but we recommend up to 20-30 active pairs per account to avoid Flood Wait from Telegram.
