Welcome to the official documentation for Content Scraper Premium.

1. Quick Start

Installation

1. Download and extract the archive.

2. Double-click install.bat to set up the environment.

Setting Up Keys

Open keys.py and enter your Telegram API credentials obtained from my.telegram.org:

API_ID = 12345678
API_HASH = 'your_hash_here'

2. Configuration

Rename .env.example to .env and configure your Bot Token:

BOT_TOKEN=123456:ABC-DEF...
ADMIN_ID=123456789

3. Admin Commands

  • /start - Open Admin Panel
  • /addpair - Add a new channel pair (Donor -> Target)
  • /list - View all active pairs
